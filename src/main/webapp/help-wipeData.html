If this option is selected, the emulator will have its user data reset at start-up, as if the emulator was newly-created.
Any modifications made to the system partition will remain untouched, as well as any SD cards and their contents that may exist.
This is equivalent to running the Android emulator command with the -wipe-data option.

Note: This option can add anywhere from 30 seconds to five minutes to the build startup time, depending on the build slave's CPU speed.
